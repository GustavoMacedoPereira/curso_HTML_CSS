FORMATAÇÃO DO TEXTO

Existem maneiras diferentes de formatar um texto para deixar ele mais legível.

Uma forma de fazer isso quando por exemplo for um código é usando a tag <code>code</code> que deixa o texto com um espaçamento melhor para leitura de códigos

Exemplo:

>>>numero1 = 5
>>>numero2 = 2
>>>print(numero1 * numero2)
>>>10

Outra forma de fazer é com a tag <pre>pre</pre> que serve para formatar o texto de acordo com o que você colocar, considerando todos os espaços

Exemplo:


    >>>numero1 = 5
    >>>numero2 = 2
    >>>print(numero1 * numero2)
    >>>10

Citações

Para fazer citações podemos usar a tag <q>q</q> de quote ( citação ) onde ela automáticamente formata o texto e coloca entre aspas

Exemplo:

Como diz o ditado Cavalo dado não se olha os dentes

Outra forma é usando citações mais longas usando a tag <blockquote>blockquote</blockquote> em que a citação é identada

Exemplo:

Esta citação é uma blockquote
