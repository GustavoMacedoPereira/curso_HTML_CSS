Listas

Listas ordenadas

Para usar listas de forma mais inteligente em HTML usamos a ta <ol>ol</ol> de Order List com a tag <li>li</li> de list item para organizar uma lista

Exemplo:

  1. Acordar
  2. Tomar café
  3. Escovar os Dentes
  4. Estuar
  5. Me arrumar
  6. Sair para o trabalho

Podemos alterar a formatação da lista dentro da tag ol

Exemplo: Com números partindo de 2

  1. Acordar
  2. Tomar café
  3. Escovar os Dentes
  4. Estuar
  5. Me arrumar
  6. Sair para o trabalho

Exemplo: Com letras Maiúsculas

  1. Acordar
  2. Tomar café
  3. Escovar os Dentes
  4. Estuar
  5. Me arrumar
  6. Sair para o trabalho

Exemplo: Com letras Minúsculas

  1. Acordar
  2. Tomar café
  3. Escovar os Dentes
  4. Estuar
  5. Me arrumar
  6. Sair para o trabalho

Exemplo: Com Números romanos maiúsculos

Listas não ordenadas

Exemplo: Usando um ponto

  • Acordar
  • Tomar café
  • Escovar os Dentes
  • Estuar
  • Me arrumar
  • Sair para o trabalho
