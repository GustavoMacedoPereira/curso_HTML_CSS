MARCA TEXTO

Para deixar uma parte do texto marcado em uma cor especial para dar ATENÇÃO ao texto usamos a tag <mark></mark>

Para alterar a cor de fundo de uma tag mark, fazemos isso dentro da própria tag: <mark style="background-color;">fundo laranja</mark>

Para saber mais sobre cores de fundo acesse developer.mozilla.org

Texto grande e texto pequeno

Para deixar um texto grande, embora não usual, podemos usar a tag <big>GRANDE</big>

E para deixar um texto pequeno para por exemplo usar no rodapé do site, usamos a tag <samll>samll</samll>

Texto excluído

Para marcar um texto como excluído, ou seja: com um risco em cima dele, usamos a tag <del>del</del>

Texto inserido

Para marcar um texto como se fosse um ponto de atenção e sublinha-lo, podemos fazer da seguinte forma:

<ins>ins</ins> que é uma tag HTML

OU

<u>u</u> que é uma tag CSS

Texto sobrescrito

<sup>sup</sup> Que coloca algum valor a cima. Exemplo: 22=4

<sub>sub</sub> Que coloca algum valor a baixo. Exemplo: H2O
